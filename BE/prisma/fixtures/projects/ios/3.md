## 📍 주요 기능

### 🧭 로컬 Apple 기기 연결

MultipeerConnectivity를 통해 근처 Apple 기기 자동 검색 및 연결

### 🧭 실시간 미러링

iPhone 카메라 화면을 iPhone/iPad/Mac으로 H.264 인코딩 스트리밍

### 🧭 타이머 자동 촬영

8초 카운트다운 후 8초 간격으로 10장 자동 촬영

### 🧭 Watch 원격제어

Apple Watch에서 촬영 버튼을 눌러 원격으로 촬영 (다른 Apple 기기로 원격 촬영 대체 가능)

### 🧭 사진 선택 및 합성

촬영한 10장 중 원하는 사진을 선택하여 다양한 프레임 레이아웃으로 합성

## 🧰 기술 스택

### 🔎 개발 환경

| 구분                      | 내용      |
| ------------------------- | --------- |
| **언어**                  | Swift 5.0 |
| **iOS Deployment Target** | iOS 17.0+ |
| **코드 품질 관리**        | SwiftLint |

### 🔎 프레임워크

| 구분             | 내용                                             |
| ---------------- | ------------------------------------------------ |
| **UI**           | SwiftUI                                          |
| **상태 관리**    | @Observable (iOS 17.0+)                          |
| **기기 간 통신** | MultipeerConnectivity                            |
| **Watch 통신**   | WatchConnectivity                                |
| **비동기 처리**  | Combine, async/await                             |
| **비디오 처리**  | AVFoundation, VideoToolbox (H.264 인코딩/디코딩) |
