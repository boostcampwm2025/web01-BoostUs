## 🔹 BoostUs는 무슨 서비스인가요?

- **BoostUs**는 부스트캠프에서 만들어진 수많은 프로젝트, 회고, 학습 경험들을 **하나로 모아주는 커뮤니티·아카이빙 플랫폼**입니다.

- 지금까지 부스트캠프의 **값진 경험**들은 개인 블로그, 깃허브, 슬랙 등 **여러 곳에 분산되어 있다는 문제점**이 있었습니다.

- BoostUs는 이 **모든 경험을 하나로 연결**해, **기록이 자산이 되고 성장이 다음 도전으로 이어지도록** 돕습니다.
